# Sentiment Analysis using NLP

## Description
This project performs sentiment analysis using Natural Language Processing techniques. It classifies text data into positive, negative, or neutral sentiments using machine learning.

## Features
- Text preprocessing
- Text vectorization using CountVectorizer
- Naive Bayes model implementation
- Sentiment prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NLP (CountVectorizer)

## Output
- Classified text into sentiments
- Tested model with custom inputs
- Analyzed customer feedback
